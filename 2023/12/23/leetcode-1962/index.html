2023/12/23 数据结构 FDS C/C++ 2023/12/23

题目

原题目

1962. 移除石子使总数最小

题目大意

每次选出数列中最大的元素, 减半, 最后求和

 

解题过程

最开始我的想法就是直接模拟, 然后时间复杂度应该是$O(kN+N)\approx O(N^2)$,我自信的觉得应该不会卡(我都不知道哪来的自信), 就直接模拟了

class Solution {
public:
    int minStoneSum(vector<int>& piles, int k) {
        while(k){
            vector<int>::iterator i,max_i;
            int max_t = 0;
            for(i = piles.begin();i!=piles.end();++i){
                if(*i>=max_t){
                    max_t = *i;
                    max_i = i;
                }
            }
            *max_i -= *max_i/2;
            --k;
        }
        int sum = 0;
        for(auto& x:piles){
            sum+=x;
        }
        return sum;
    }
};

结果不出所料, TLE了, 于是换用优先队列, 分析时间复杂度, 主要在建堆, 处理, 求和三个阶段: $O(N\log N+k\log N+N)\approx O(N\log N)$, 确实比原先的$O(N^2)$少不小.

但是使用优先队列涉及到几个问题:

  • 如何用vector初始化一个priority_queue: 使用始末迭代器
  • 如何获取堆顶的元素: pq.top()
  • 如何删除堆顶的元素: pq.pop()

解决完以上问题, 我们就可以给出代码了

class Solution {
public:
    int minStoneSum(vector<int>& piles, int k){
        priority_queue<int> Piles(piles.begin(), piles.end());
        while(k){
            int tmp = Piles.top();
            Piles.pop();
            tmp -= tmp/2;
            Piles.push(tmp);
            --k;
        }
        int ret = 0;
        while(Piles.empty()==false){
            ret+=Piles.top();
            Piles.pop();
        }
        return ret;
    }
};

提交结果, AC, 但是时空成绩并不是很优

三体人语言

注意python的heapify函数没有大根堆, 因此只能取反之后生成小根堆当作大根堆用

class Solution:
    def minStoneSum(self, piles: List[int], k: int) -> int:
        heap = [-a for a in piles]
        heapify(heap)
        for i in range(k):
            a = -heappop(heap)
            a = a - a//2
            heappush(heap, -a)
        return -sum(heap)

手写堆嘛, 有什么大不了的

void swap(int *nums, int i, int j) {
    int x = nums[i];
    nums[i] = nums[j];
    nums[j] = x;
}

void down(int *nums, int size, int i) {
    for (int k = 2 * i + 1; k < size; k = 2 * k + 1) {
        // 父节点 (k - 1) / 2，左子节点 k，右子节点 k + 1
        if (k + 1 < size && nums[k] < nums[k + 1]) {
            k++;
        }
        if (nums[k] < nums[(k - 1) / 2]) {
            break;
        }
        swap(nums, k, (k - 1) / 2);
    }
}

void Init(int *nums, int size) {
    for (int i = size / 2 - 1; i >= 0; i--) {
        down(nums, size, i);
    }
}

void Push(int *nums, int size, int x) {
    nums[size] = x;
    for (int i = size; i > 0 && nums[(i - 1) / 2] < nums[i]; i = (i - 1) / 2) {
        swap(nums, i, (i - 1) / 2);
    }
}

int Pop(int *nums, int size) {
    swap(nums, 0, size - 1);
    down(nums, size - 1, 0);
    return nums[size - 1];
}

int minStoneSum(int* piles, int pilesSize, int k) {
    int *pq = piles, n = pilesSize;
    Init(pq, n);
    for (int i = 0; i < k; i++) {
        int pile = Pop(pq, n);
        n--;
        pile -= pile / 2;
        Push(pq, n, pile);
        n++;
    }
    int sum = 0;
    while (n > 0) {
        sum += Pop(pq, n);
        n--;
    }
    return sum;
}
