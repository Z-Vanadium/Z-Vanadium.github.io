棋局

2023/12/9 C/C++

「苍天如圆盖，陆地似棋局。世人黑白分，往来争荣辱。荣者自安安，辱者定碌碌。」
——罗贯中《三国演义·第三十七回》

Pac 善观万物往复，而小小棋盘有如世间缩影，尤为她所喜爱。今天闲来无事，她又发明了一种新的棋。

给出一个有 N 个格子水平排列的棋盘，从左到右标号为 1 到 N ，游戏的开始时有 M 个棋子，标号为 1,⋯,M，标号为 i 的棋子在格子 $a_i$ 上。

棋子可以重叠，我们默认标号小的棋子位于标号大的棋子上面。

现有 K 张行动卡片，行动卡片上指定了移动棋子的标号 a 和位移 b，使用卡片会使得标号 a 的棋子向右移动 b 格（注意位移可能为负值）。

当棋子移动到一个已经存在其他棋子的格子上时，移动的棋子会叠在目标位置所有棋子的上方，一个棋子移动时会使得其上方的所有棋子一起移动。 现在按照给出卡片的顺序依次进行操作，请输出操作结束后所有棋子所在的格子的位置。

输入格式:

第 1 行三个整数 N,M,K， 表示棋盘的大小，棋子的数量和行动卡片的数量。

第 2 行 M 个整数 $a_1$ , $a_2$ ,⋯, $a_M$ 表示棋子的初始位置。

第 3 行到第 2+K 行，每行两个整数 a, b 描述一张行动卡片，分别表示移动的棋子标号和位移。

保证对于所有用例，每次行动不会超出棋盘的范围。

输出格式:

共一行 M 个整数，用空格隔开，第 i 个整数表示标号为 i 的棋子的最终位置。

样例:

输入样例 1

9 3 12
2 3 4
1 1
1 1
1 2
2 1
2 1
1 -1
3 1
2 2
3 -1
2 -1
3 1
3 2

输出样例1

9 9 9

数据范围

对于所有测试点，满足 1 ≤ N ≤ $10^3$ ,1≤M≤50,1≤K≤$10^4$ 。

对于20%的数据，M=1。

代码长度限制                    16 KB
时间限制                        400 ms
内存限制                        64 MB

解题过程

这道题应该难度不大，思路大概是开一个一维数组，每个元素储存一个链表，用来维护棋子的状态，因为棋子可以集体移动，所以链表的表示是最好的。
思路没有什么大问题，可惜最后没有AC，在群里面学习之后，分析原因应该是没有额外储存每个棋子的位置，导致部分样例点过不去。
下面是120/300的代码，现场写的，可读性有亿点差>_<

#include<iostream>
using namespace std;

typedef struct node* pnode;
struct node {
    int val;
    struct node* next;
};
int N,K,M;
pnode map[1005];

pnode locate(int x,int* where){
    for(int i=1;i<=N;++i){
        if(map[i]!=NULL){
            pnode tmp = map[i];
            if(tmp->val == x){
                map[i] = NULL;
                *where = i;
                return tmp;
            }
            else{
                pnode pre = map[i];
                pnode cur = map[i]->next;
                while(cur!=NULL){
                    if(cur->val == x){
                        pre->next = NULL;
                        *where = i;
                        return cur;
                    }
                    pre = cur;
                    cur = cur->next;
                }
            }
        }
    }
    return NULL;
}

int main(){
    cin>>N>>M>>K;
    for(int i = 1;i<=M;++i){
        pnode tmp = new node;
        tmp->val = i;
        tmp->next = NULL;
        int a;
        cin>>a;
        if(map[a]==NULL){
            map[a]=tmp;
        }
        else{
            tmp->next = map[a];
            map[a] = tmp;
        }
    }
    for(int i=1;i<=K;++i){
        int a,b;
        int newa;
        cin>>a>>b;
        pnode pre = locate(a,&newa);
        newa += b;
        pnode k = map[newa];
        if(k == NULL){
            map[newa] = pre;
        }
        else{
            while(k->next!=NULL){
                k=k->next;
            }
            k->next = pre;
        }
    }

    for(int i=1;i<=M;++i){
        int p;
        locate(i,&p);
        cout<<p<<" ";
    }
}
