ロード・エルメロイII世の事件簿

2023/12/9 C/C++ 2023/12/09

从第四次圣杯战争幸存后，韦伯·维尔维特在试图环球旅行结果遭受到世界的不幸对待：抢劫，暴力冲突······心灰意冷的韦伯决定回到他所熟悉的钟塔成为讲师维生。

在一次机缘巧合之下，韦伯受Reines·El-Melloi·Archisorte(莱涅丝·埃尔梅罗)所要挟，继任了”Lord·El-Melloi”的称号。但出于对上一代El-Melloi的尊敬与愧疚，他一直自称，并要求他人称呼他为”Lord·El-MelloiⅡ”，并且担任现代魔术科的学部长。

尽管作为魔术师的韦伯实力二流，但作为讲师，他具有令人瞠目的知识量～～，以及作为侦探，他具有令人瞠目的分析、解体能力，which 令他树敌无数～～。

当当当～～～当钟塔的下课铃响起，大本钟·伦敦之星$\star$君主·埃尔梅罗Ⅱ世完成了今日的课程，作为现代魔术科的一名学生，今天所需要完成的作业是解密一段卢恩符文串！幸运的是这段卢恩符文串已经用字母谐音表示，并且已经找到了若干个字典字符串辅助翻译。那么只需要进行一一对应，然后翻译就行了。

来自资历最长的学生——斯芬的tips：

  1. 谐音后的卢恩符文串是由若干个大写字母组成的字符串；
  2. 各字典字符串由一个卢恩符文串的谐音以及一个对应字符表示,且各字典串不为其他字典串的子串。

努力完成今天的作业吧!未来的魔术之星。

输入格式

第一行一个字符串 S，表示作为作业的卢恩咒语原字符串。

第二行一个整数 n，表示有 n 个字典字符串。

第三至 2n+2 行, 第 2i+1 行（i=1,2,⋯n）为字典字符串 i 的谐音段 $s_i$ ；第 2i+2 行（i=1,2,⋯,n ）为字典字符串 i 的对应字符 $ch_i$ 。

输出格式

一行一个字符串，表示翻译后的作业答案。

样例

输入

YYSWYSYQYQBYO
4
YYSWYS
@
YQYQBYO
w
MPPPAOWND
I
TIHPASN
P

输出

@w

说明&提示

本题采用ASCII字符集。

对于100%的数据:∣S∣⩽100,1⩽n⩽30,∣ $s_i$ ∣⩽∣S∣。

代码长度限制                    100 KB
时间限制                        1500 ms
内存限制                        256 MB

解题过程

背景长的一般是水题
因为题目给的时空限制太宽裕了，因此可以直接暴力AC，可以用双指针的做法，也可以随意模拟，下面给出双指针做法

#include<bits/stdc++.h>
using namespace std;

string input;
int n;
typedef struct dictnode_ dictnode;
struct dictnode_{
    string key;
    char val;
};
dictnode dict[35];

bool check(int left, int right){
    string tmp = input.substr(left,right-left+1);
    for(int i=0;i<n;++i){
        if(tmp == dict[i].key){
            cout<<dict[i].val;
            return true;
        }
    }
    return false;
}

int main(){
    cin>>input;
    cin>>n;
    for(int i=0;i<n;++i){
        cin>>dict[i].key>>dict[i].val;
    }
    int left, right;
    left = right = 0;
    while(input[left] != '\0'){
        if(check(left,right)){
            ++right;
            left = right;
        }
        else{
            ++right;
        }
    }
}
