文章格式规范

发表于2023-12-28|更新于2024-03-27|文档
|阅读量:

记录本站文章书写格式规范

层次规范

标题使用

小标题按层次使用, 从三级标题(也可从二级标题)开始, 依次向下使用

# / <h1>: 一级标题, 仅用于文章标题
## / <h2>: 二级标题, 一般不用
### / <h3>: 三级标题
#### / <h4>: 四级标题

** ** / <strong>: 用于四级标题下, 不适合用分点处
<p>: 正文
- <ul><li>: 四级标题下分点

  • 分层级要求的笔记, 使用" Lecture d - d.d - d.d.d "的格式组织, 分别使用 2, 3, 4 级标题
  • 对于有高分层级要求的笔记, 使用" Lecture dd - d - d.d - d.d.d "的格式组织, 分别使用 1, 2, 3, 4 级标题, 即每个 Lecture 中均包含一份或多份独立的笔记
  • 英文书写笔记时, 章节标题应该遵循英文大小写书写规范

 

换行使用

四级及以上的小标题所引段落之间需使用换行符换行隔断

&nbsp; 换行符

 

分割线使用

-– / <hr>: 划分文章层次使用

 

内容规范

代码使用

` ` / <code>: 代码行, 用于语句,变量,函数等来自或部分来自于程序源码的文字
``` ``` / <pre>: 代码块, 用于成段的程序源码, 注意标记语言, 统一使用以下标记符(持续补充)

标记符 语言
c C
C++ C++
css CSS(层叠样式表)
html HTML(超文本标记语言)
js JavaScript
bash (new), line (old) 命令行
plaintext 伪代码, 自然语言
python Python
md Markdown

 

解释引用使用

> / <blockquote>: 用于不影响正文阅读的情况下, 对前文进行解释或引用一段文字

 

外链使用

[]() / <a>: 链接, 规范格式为[引用来源 | 文献标题], 也可在不影响阅读的情况下, 镶嵌在文章内部, 颜色#66afef

例如: 百度百科 | c语言

 

公式使用

$$: 出现中文, 一起放入公式行中

Katex 公式规范

  • 表示变量或常量的字母, 应当是斜体, 如x,y,zx,y,zx,y,z
  • 表示函数名, 算子的字母, 应当是正体\mathrm{}, 如d,Δ,max{}\mathrm{d},\mathrm{\Delta},\mathrm{max}\lbrace\rbraced,Δ,max{}
  • 表示向量的字母, 应当是粗体\bm{}, 如α,β\bm{\alpha},\bm{\beta}α,β
  • 由于 Katex 编译渲染机制, {}需使用\lbrace\rbrace
