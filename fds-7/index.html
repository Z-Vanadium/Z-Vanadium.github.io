Chapter 6 Graph

Definition

  • G(V, E)
    • G ::= graph
    • V ::= vertice
    • E ::= edge
  • Undirected graph: <v,w>=<w,v><v, w> = <w, v><v,w>=<w,v>
  • Directed graph: <v,w>≠<w,v><v, w> \ne <w, v><v,w>=<w,v>
  • Restrictions:
    • No self loop
    • No multigraph
  • Complete graph: has the maximum number of edges
    • Undirected Graph: V=n→E=Cn2V = n\to E=C^2_nV=n→E=Cn2​
    • Directed Graph: V=n→E=Pn2V = n\to E=P^2_nV=n→E=Pn2​
  • Adjacent:
    • Adjacent to
    • Adjacent from
  • Subgraph
  • Path form vpv_pvp​ to vqv_qvq​
  • Length of path
  • Simple path: No repeated viv_ivi​
  • Cycle: simple path with vp,vqv_p,v_qvp​,vq​
  • Connected
  • Component of an undirected graph: the maximal connected subgraph
  • Tree: connected & acyclic graph
  • DAG: directed acyclic graph
  • Strongly connected directed graph
    • Weakly conncted directed graph (ignore direction)
  • Strongly connected component
  • Degree(v): number of edges incident to v
    • in-degree
    • out-degree
    • e=(∑i=0n−1di/2)e=(\displaystyle\sum_{i=0}^{n-1} d_i/2)e=(i=0∑n−1​di​/2)
