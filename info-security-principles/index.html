什么是信息？

信息

  • 广义：Information is data endowed with relavance and purpose.

  • 狭义：信息的本质是有序度。——Shannon

信息量：对某件事情可供选择的多少的量度

香农信息量：log⁡2N\log_2Nlog2​N

 

什么是安全？

安全定义

  1. 第一步：针对谁？
    • 谁是敌手（adversary）？
  2. 第二步：敌手的目标
    • 敌手想要达到什么目标
  3. 第三步：限制敌手的能力
    • 敌手可以做什么？不可以做什么？
    • 敌手可以调用的资源
  4. 第四步：如何认定敌手获胜？如何认定系统被破解？
    • 敌手挑战的时间
    • 敌手达成目标的概率

如果没有敌手可以在规定的时间内攻破该系统，那么说系统是安全的

假如有人让你帮忙测试某系统是否安全，请反问ta：什么是安全？
