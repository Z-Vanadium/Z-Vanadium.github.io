3.2.2 Complement code

Eg in 12 o’clock

$10 - 4 = 6$

$10 + 8 = 18 \equiv 6 ( \mathrm{mod}\ 12 )$

Definition $\forall X_T: [X_T]_C=M+X_T(\mathrm{mod}\ M)$

The Conversion between Complement code and true value

  • If the sign bit is 0 / the number is positive:
    The value part does not need to be changed
  • Else the sign bit is 1 / the number is negative:
    Invert the bits of value part, and add 1 to the result
  • From $[X_T]_C$ to $[-X_T]_C$:
    Invert all the bits, and add 1 to to the result
