2 位移 速度 加速度

2.1 位矢

$\bm{r}=x\bm{l}+y\bm{j}+z\bm{k}$

$r=|\bm{r}|=\sqrt{x^2+y^2+z^2}$

$\cos\alpha=x/r$
$\cos\beta=y/r$
$\cos\gamma=z/r$

 

2.2 矢量的加法和乘法

2.2.1 点乘

$\bm{a}\cdot\bm{b}=a\times b\times\cos\theta$

2.2.2 叉乘

 

2.3 位移

$\Delta\bm{r}=\bm{r}_{AB}=\bm{r}_B-\bm{r}_A$

 

2.4 速度

2.4.1 平均速度

$\bar{\bm{v}}=\displaystyle\frac{\Delta\bm{r}}{\Delta t}$

2.4.2 平均速率

$\bar{v}=\displaystyle\frac{\Delta{s}}{\Delta t}$

2.4.3 瞬时速度

$\bm{v}=\displaystyle\lim_{\Delta t\rightarrow0}\frac{\bm{r}(t+\Delta t)-\bm{r}(t)}{\Delta t}=\lim_{\Delta t\rightarrow0}\frac{\Delta\bm{r}}{\Delta t}=\frac{\mathrm{d}\bm{r}}{\mathrm{d}t}$

 

2.5 加速度

2.5.1 直角坐标系下

$a_y=\displaystyle\frac{\mathrm{d}^2y}{\mathrm{d}t^2}$

$a_z=\displaystyle\frac{\mathrm{d}^2z}{\mathrm{d}t^2}$

3 自然坐标系与运动

3.1 圆周运动及其描述

3.1.1 自然坐标系描述

$\bm{v}=v_t\bm{e}_t$

$\bm{a}=\displaystyle\frac{\mathrm{d}\bm{v}}{\mathrm{d}t}=\frac{\mathrm{d}v}{\mathrm{d}t}\bm{e}_t+v\frac{\mathrm{d}\bm{e}_t}{\mathrm{d}t}$

$其中\displaystyle\frac{\mathrm{d}\bm{e}_t}{\mathrm{d}t}=\frac{\bm{e}^\prime_t-\bm{e}_t}{\Delta t}=\frac{\Delta\theta\cdot\bm{e}_t}{\Delta t}=\frac{1}{R}v\bm{e}_t$

$\Rightarrow 向心加速度 a_n=\displaystyle\frac{v^2}{R}$

$(\theta\backsim s,R为曲率半径)$

3.1.2 角量描述

规定逆时针转动为正

平面圆周运动转化为一维运动

$v=R\omega$

 

3.2 抛体运动

$\bm{r} = \displaystyle\int_0^t\bm{v}\mathrm{d}t=(v_0t\cos\theta)\bm{l}+(v_0t\sin\theta-\frac{1}{2}gt^2)\bm{j}$

 

3.3 相对运动

3.3.1 伽利略变换

$设参考系S^\prime相对S向+x运动，速度为u(经典物理学)，有：$

$x^\prime = x - ut$
$y^\prime = y$
$z^\prime = z$
$t^\prime = t$

$\Rightarrow两个惯性系中,\bm{a}=\bm{a}^\prime$

 

Lecture 02

1 牛顿运动定律

1. 1 牛顿第一定律（惯性定律）

惯性参考系：惯性定律成立的参考系

强行设置的，回环定义了属于是

 

1.2 牛顿第二定律

$$\bm{F}=m\bm{a}$$

$m$: 惯性质量（未来还有很多种质量需要区分）

牛二不能包含牛一，即使牛二被证伪（相对论），牛一也可以是正确的

1.2.1 叠加原理

$$\bm{F}=\sum\bm{F}_i$$

1.2.2 微分形式: 动量定理

$$\bm{p}=m\bm{v}$$

在微观高速情况下依然适用

 

1.3 牛顿第三定律

  • 力不是凭空出现的，是相互成对的
  • 与动量守恒互为前提和结论

 

2 常见力与基本力

2.1 重力

忽略自转 $g=g_0=\displaystyle\frac{G_0M}{R^2}$

考虑自转 $g=g_0(1-\displaystyle\frac{\cos^2\varphi}{290})$

290 是一个和自转轴倾斜度有关系的参数，一般不考虑

2.2 弹力

2.2.1 弹簧的弹力

$\bm{F}=-k\bm{x}$

2.3 摩擦力

绞盘模型

$T_B=T_Ae^{-\mu\varTheta}$

$\Rightarrow T_B\ll T_A$

 

2.3 四种基本力

2.3.1 万有引力

牛顿万有引力定律

$$F=G_0\displaystyle\frac{m_1m_2}{r^2}$$

$m$: 引力质量

如果愿意，可以将 $\sqrt{G_0}m$ 定义为引力荷 $Q$

引力质量有别于惯性质量，本可以是完全不一样的。但是仅仅是伽利略的自由落体实验发现了这两者的关联性，最终导致了爱因斯坦的广义相对论（这一切都是命运石之门的安排）

均匀球壳对内部的万有引力为 0

$\displaystyle\frac{\mathrm{d}x^2}{\mathrm{d}^2t}=-\omega^2x\Rightarrow x=A\cos\omega t$
