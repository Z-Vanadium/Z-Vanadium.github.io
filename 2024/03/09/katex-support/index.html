KaTex公式渲染相关

2024/3/9 实用 Hexo

由于 Hexo 主题 ParticleX 对 Markdown 的渲染和 KaTex 对 Tex 公式的解释之间很容易产生冲突, 这里记录一些冲突的处理办法同时, 记录一些可能用到的或" 书到用时方恨少 "的 Tex 公式关于 KaTex 公式的书写规范, 可另见

前言

Hexo 版本: 6.3.0
Hexo 主题: ParticleX

 

冲突

下划线斜体与下标冲突

当使用_表示下标的时候, 如果下标使用了{}扩起并使用了 2 个及以上的下划线, 就有可能使下划线被解释为斜体, 导致公式无法正常渲染

解决方法: 使用 0 长度空格\kern替代{}

实例

$(100101.01)_2=2^5+2^2+2^0+2^{-2}=(37.25)_1\kern{}_0$

$(100101.01)_2=25+22+20+2{-2}=(37.25)_1\kern{}_0$
