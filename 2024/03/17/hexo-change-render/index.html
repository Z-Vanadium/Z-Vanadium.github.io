Hexo 修改渲染器为 hexo-renderer-markdown-it 以及一些新的语法

发表于2024-03-17|更新于2024-03-23
|阅读量:

折腾了一个小时, 给 Hexo 修改了渲染器

 

1 安装步骤

1.1 卸载原有的渲染器

进入 Hexo 根目录, 也就是包含了 package.json 和 package-lock.json 两个配置文件的路径, 运行以下命令卸载默认的渲染器

npm uni hexo-renderer-marked --save

1.2 安装新的渲染器

运行以下命令安装新的渲染器

npm i hexo-renderer-markdown-it --save

1.3 修改配置文件

将以下文本追写到 Hexo 的配置文件 _config.yml 后面

markdown:
  preset: "default"
  render:
    html: true
    xhtmlOut: false
    langPrefix: "language-"
    breaks: true
    linkify: true
    typographer: true
    quotes: "“”‘’"
  enable_rules:
  disable_rules:
  plugins:
    - markdown-it-abbr
    - markdown-it-cjk-breaks
    - markdown-it-deflist
    - markdown-it-emoji
    - markdown-it-footnote
    - markdown-it-ins
    - markdown-it-mark
    - markdown-it-sub
    - markdown-it-sup
    - markdown-it-checkbox
    - markdown-it-imsize
    - markdown-it-expandable
    - name: markdown-it-container
      options: success
    - name: markdown-it-container
      options: tips
    - name: markdown-it-container
      options: warning
    - name: markdown-it-container
      options: danger
  anchors:
    level: 2
    collisionSuffix: ""
    permalink: false
    permalinkClass: "header-anchor"
    permalinkSide: "left"
    permalinkSymbol: "¶"
    case: 0
    separator: "-"

具体每一项的意思可以参考原项目: https://github.com/hexojs/hexo-renderer-markdown-it

1.4 安装其他插件

npm i markdown-it-checkbox
npm i markdown-it-imsize
npm i markdown-it-expandable

1.5 修改主题的 CSS 文件

修改主题的 CSS 文件, 使新的语法对应的内容符合你的主题风格

 

2 新的语法

2.1 基础语法

  1. 下标

    H~2~O

    H2O

  2. 上标

    X^2^

    X2

    Katex 失业了一点点, 但不多

  3. 复选框

    未选：- [ ]
选中：- [x]


  4. 高亮

    ==高亮==

    高亮

不要总结
+++
